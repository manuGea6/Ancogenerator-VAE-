# TP AnCoGen – Notebook corrigé

Ce dossier contient une version **corrigée et exécutable** du notebook étudiant `ancogen_lab_student.ipynb`.

## Contenu
- `ancogen_lab_solution.ipynb` : notebook complété (téléchargement des modèles, chargement GPU, outils, `analysis`, `synthesis`, `snr_control`)
- `requirements.txt` : dépendances Python (pour exécution locale si souhaitée)

## Exécution (Colab recommandé)
1. Ouvrir `ancogen_lab_solution.ipynb` dans Google Colab
2. Exécuter les cellules dans l'ordre :
   - Installation des dépendances
   - Téléchargement des modèles depuis Hugging Face
   - Chargement des modèles sur GPU/CPU
   - Tests / analyses / synthèses

## Notes
- Les chemins `/content/...` sont ceux utilisés par défaut dans Colab.
- Si vous exécutez en local, adaptez les chemins en conséquence.
